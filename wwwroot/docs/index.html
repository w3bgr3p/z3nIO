z3nIO

z3nIO — локальный веб-дашборд для управления фермой аккаунтов на базе ZennoPoster. Запускается как отдельный .NET 8 процесс на Windows-машине оператора и открывается в браузере.

Дашборд не заменяет ZennoPoster — он работает поверх него: читает состояние задач из БД, отправляет команды в очередь, принимает логи и трафик от запущенных ZP-шаблонов.

Что делает

Управление задачами — запуск, остановка, настройка ZP7-задач без открытия ZennoPoster. Счётчики, потоки, расписание — всё через браузер.

Мониторинг — живые логи и HTTP-трафик от каждой задачи в реальном времени через SSE.

Инструменты разработки — перехват и replay HTTP-запросов, JSON-viewer с детектированием токенов и captcha, кодогенерация (C#, Python, cURL).

Автоматизация — встроенный планировщик для запуска .py / .js / .exe / .bat скриптов по cron, интервалу или фиксированному времени.

Хранилище шаблонов — Cliplates: дерево copy-paste сниппетов, копирование одним кликом.

Как запускается

При старте z3nIO читает appsettings.secrets.json. Если файл отсутствует — дашборд открывается на странице shift + ctrl + 0. Config для первоначальной настройки. Если файл есть — сразу открывается shift + ctrl + 1. z3nIO (Scheduler).

Дашборд доступен по адресу http://localhost:10993 (порт настраивается в shift + ctrl + 0. Config).

Архитектура с точки зрения оператора

ZennoPoster задачи
    │  логи → LogHost
    │  трафик → TrafficHost
    │  команды ← БД (_commands)
    ▼
z3nIO (встроенный HTTP-сервер)
    │
    ▼
Браузер (дашборд)

ZP-шаблоны отправляют логи и трафик на адреса LogHost / TrafficHost из конфига. z3nIO принимает их и сохраняет в БД. Команды (start, stop, и др.) z3nIO кладёт в таблицу _commands — ZP-шаблон опрашивает её и выполняет.

Страницы

shift + ctrl + 1. z3nIO (Scheduler)

Запуск скриптов по расписанию.

  • Список задач — группировка, статусы, executor-фильтр
  • Создание задачи — имя, путь, executor, args, enabled
  • Расписание — cron, interval, fixed time, overlap policy
  • Execution tab — статус, PID, uptime, memory, parallel instances
  • Output tab — вывод последнего запуска
  • Payload — Set Values, Build Schema, Import, Export
  • Logs панель — логи задачи + SSE live
  • HTTP панель — трафик задачи + SSE live

shift + ctrl + 2. ZP7

Управление ZP7-задачами. Центральная страница фермы.

  • Список задач — группировка machine → project, статусы, индикаторы
  • Фильтры и теги — поиск по имени, статусу, GroupLabels
  • Task - Detail — карточка выбранной задачи
  • Команды — start, stop, interrupt, tries, threads
  • Execution Settings — потоки, приоритет, счётчики, прокси, GroupLabels
  • Scheduler Settings — period, intervals, start/end, repeat
  • Settings — редактор полей InputSettings задачи
  • Logs панель — логи задачи + SSE live
  • HTTP панель — трафик задачи + SSE live
  • Commands Queue — очередь команд, фильтр, очистка
  • Heatmap — визуализация успех/ошибка по проекту

shift + ctrl + 3. ZB

Мониторинг и управление ZennoBrowser через ZB API.

  • Статус подключения — host, key, connect
  • Instances — запущенные инстансы, Stop / Kill, WS Endpoint
  • Profiles — список профилей, поиск, сортировка, запуск
  • Proxies — прокси и статус проверки
  • Threads — активные потоки, освобождение

shift + ctrl + 4. Logs

Живые логи приложения.

  • Таблица логов — Time, Level, Machine, Project, Uptime, PID, Port, Account, Origin, Message
  • Фильтры — level, machine, project, session, port, pid, account, origin, uptime
  • Detail-панель — полная запись, копирование сообщения
  • Счётчики — shown, errors, warnings
  • Edge cases — логи не появляются, uptime не считается

shift + ctrl + 5. HTTP

Перехваченный HTTP-трафик из ZP-задач.

  • Список запросов — method, status, url, duration, account
  • Фильтры — method, status, URL, machine, project, limit
  • Detail-панель — request headers/cookies/body + response headers/body
  • Replay — повторная отправка запроса с редактированием
  • Кодогенерация — HttpClient, ZP7, Hybrid, Python, TypeScript, cURL
  • Example — агрегация endpoint’ов из текущей выборки

shift + ctrl + 6. JSON

Интерактивный JSON-viewer с security-детектированием.

  • Ввод и парсинг — вставка JSON, авто-парсинг, Ctrl+Enter
  • Навигация по дереву — collapse/expand, depth-кнопки, виртуальный скролл
  • Security Badges — Bearer, Basic, API Key, CF, Turnstile, Captcha
  • Field Filter — фильтр по полям массива, пресеты api / headers / status
  • Поиск — подстрока по ключам и значениям
  • Действия на узлах — copy value, C# selector, replay URL, hide/show
  • Replay — отправка запроса из JSON-структуры

shift + ctrl + 8. Clips

Дерево copy-paste шаблонов.

  • Дерево — папки по path, навигация, поиск
  • Копирование — клик по листу → буфер обмена
  • Редактор — создание, редактирование, удаление записи

shift + ctrl + 7. Text Tools

Конвертеры строк.

  • Decode
  • [[shift + ctrl + 7. Text Tools#C# String Escaper|C# String Escaper]] — обычный и verbatim (@"...")
  • Decode
  • JSON String Escape

shift + ctrl + 0. Config

Настройки сервера и подключений.

  • Status — uptime, порт, режим БД, хосты, listening ports
  • DbConfig — SQLite или PostgreSQL
  • Logs & Server — Dashboard Port, LogHost, TrafficHost, папки, MaxFileSize
  • jVars — PIN, путь к .dat-файлу, расшифровка кошельков
  • Storage — размер лог-файлов, прогресс-бар, список файлов
  • Очистка логов — App logs, HTTP logs, Traffic logs, ALL logs
  • Fill DB — загрузка аккаунтов, прокси и данных фермы
